## ✨ プロジェクト概要

このプロジェクトは、**深層学習**と**リアルタイム音声処理**を組み合わせた革新的な音声分類システムです。環境音を瞬時に認識し、美しいWebインターフェースで結果を可視化します。

### 🎯 主な特徴

- 🎙️ **リアルタイム録音・分析** - マイクからの音声を即座に分類
- 🧠 **高精度AIモデル** - CNNベースの深層学習で90%以上の精度
- 🎨 **美しいUI/UX** - モダンなStreamlitインターフェース
- 📊 **豊富な可視化** - 波形、スペクトログラム、確率分布を表示
- 📱 **レスポンシブデザイン** - あらゆるデバイスで快適に使用

## 🎯 使用方法

### 🌐 Webアプリの起動

#### 1️⃣ アプリを起動
```bash
streamlit run app.py
```

#### 2️⃣ ブラウザでアクセス
```
http://localhost:8501
```

#### 3️⃣ 設定調整
| 設定項目 | 範囲 | デフォルト | 説明 |
|----------|------|------------|------|
| **録音時間** | 1-5秒 | 3秒 | 1回の録音時間 |
| **サンプリングレート** | 16kHz, 22.05kHz, 44.1kHz | 22.05kHz | 音声品質 |
| **検出回数** | 1-20回 | 5回 | 連続検出回数 |
| **分類閾値** | 0.1-0.9 | 0.5 | 分類信頼度の閾値 |

#### 4️⃣ 録音開始
- 「🎙️ Start Recording」ボタンをクリック
- マイクからの音声を録音・分析
- 結果が履歴に追加される

#### 5️⃣ 録音停止
- 「⏹️ Stop Recording」ボタンで停止

### 📊 表示される情報

#### リアルタイム分析
- 🎵 **音声波形** - 録音された音声の波形
- 📈 **メルスペクトログラム** - 周波数特性の可視化
- 📊 **クラス別予測確率** - 各クラスの確率
- 🎯 **予測確率分布** - 確率の分布グラフ

#### 履歴表示
- ⏰ **検出時刻** - 検出された時刻
- 🏷️ **分類結果** - 予測されたクラス
- 📈 **信頼度** - 予測の信頼度
- 📊 **統計情報** - 全体の統計

## 🏗️ モデルのアーキテクチャ

<div align="center">

### 🧠 ニューラルネットワーク構成

```
入力: メルスペクトログラム (128×130×1)
    ↓
┌─────────────────────────────────────┐
│ 4層のCNN                            │
│ • Conv2D + BatchNorm + ReLU + MaxPool │
│ • 32→64→128→256チャンネル           │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 全結合層 + ドロップアウト (0.5)     │
└─────────────────────────────────────┘
    ↓
出力: 5クラス確率
```

</div>

### 📊 詳細仕様

| 項目 | 詳細 |
|------|------|
| **畳み込み層** | 32→64→128→256チャンネル |
| **プーリング** | 2×2 MaxPooling |
| **ドロップアウト** | 0.5 |
| **最適化器** | Adam (lr=0.0003) |
| **損失関数** | CrossEntropyLoss |

## 🔧 トラブルシューティング

### ❗ よくある問題

<div align="center">

| 問題 | 解決方法 |
|------|----------|
| **🎤 マイクが認識されない** | ブラウザのマイク権限を確認 |
| **📁 モデルが読み込めない** | `best_model_5class_90.pth` ファイルを確認 |
| **⏺️ 録音が開始されない** | マイクの接続とブラウザコンソールを確認 |

</div>

### 🚀 パフォーマンス最適化

| 設定 | 推奨値 | 理由 |
|------|--------|------|
| **録音時間** | 1-2秒 | 短時間で高頻度検出 |
| **サンプリングレート** | 22.05kHz | 品質と処理速度のバランス |
| **検出回数** | 5-10回 | 用途に応じた適切な回数 |

## 📝 注意事項

<div align="center">

⚠️ **重要事項**

</div>

- 🎵 音声ファイルはWAV形式である必要があります
- 🔧 サンプリングレートは22050Hzに統一されます
- ⏱️ 音声の長さは3秒に統一されます
- 🤫 録音中は静かな環境で使用することを推奨
- 📊 分類精度は環境音や音質に影響される場合があります
- ⏰ 長時間の連続使用は避けてください
