# 🎵 リアルタイム音声分類システム

Streamlitを使用したリアルタイム音声分類Webアプリケーションです。

## 🚀 機能

- **リアルタイム録音**: マイクからの音声をリアルタイムで録音
- **音声分類**: 5つの音声クラス（犬、猫、鳥、指パッチン、人の声）を自動分類
- **可視化**: 波形、メルスペクトログラム、予測確率を視覚的に表示
- **統計情報**: 予測履歴と精度の統計表示
- **設定可能**: 録音時間、サンプリングレート、分類閾値を調整可能

## 📋 必要な環境

- Python 3.8以上
- マイク（音声入力用）

## 🛠️ インストール

1. 依存関係をインストール:
```bash
pip install -r requirements.txt
```

2. モデルファイルが存在することを確認:
   - `best_model_5class_90.pth` が同じディレクトリにあることを確認

## 🎯 使用方法

1. アプリを起動:
```bash
streamlit run app.py
```

2. ブラウザで `http://localhost:8501` にアクセス

3. サイドバーで設定を調整:
   - 録音時間（1-5秒）
   - サンプリングレート（16kHz, 22.05kHz, 44.1kHz）
   - 分類閾値（0.1-0.9）

4. 「🎙️ 録音開始」ボタンをクリックして録音開始

5. 「⏹️ 録音停止」ボタンで録音停止

## 📊 表示内容

### リアルタイム分析
- **音声波形**: 録音された音声の波形表示
- **メルスペクトログラム**: 音声の周波数特性を時間軸で表示
- **予測確率**: 各クラスに対する予測確率を棒グラフと円グラフで表示
- **検出結果**: 分類結果と確率をカード形式で表示

### 統計情報
- 総予測回数
- 最近10回の精度
- 検出された音声の分布グラフ

## 🎵 対象音声クラス

1. **dog** - 犬の鳴き声
2. **cat** - 猫の鳴き声
3. **chirping_birds** - 鳥のさえずり
4. **finger_snap** - 指パッチン
5. **human_voice** - 人の声

## ⚙️ 技術仕様

- **フレームワーク**: Streamlit
- **音声処理**: librosa, sounddevice
- **機械学習**: PyTorch
- **可視化**: matplotlib
- **データ処理**: numpy, pandas

## 🔧 トラブルシューティング

### マイクが認識されない場合
- ブラウザのマイク権限を確認
- システムの音声設定を確認

### モデルが読み込めない場合
- `best_model_5class_90.pth` ファイルが存在することを確認
- ファイルパスが正しいことを確認

### 録音が開始されない場合
- マイクの接続を確認
- 他のアプリケーションがマイクを使用していないことを確認

## 📝 注意事項

- 録音中は静かな環境で使用することを推奨
- 分類精度は環境音や音質に影響される場合があります
- 長時間の連続使用は避けてください

## 🎨 カスタマイズ

アプリの見た目や機能をカスタマイズしたい場合は、`app.py` の以下の部分を編集してください：

- CSSスタイル: `st.markdown` 内の `<style>` タグ
- レイアウト: `st.columns` の配置
- 可視化: `create_visualization` 関数
- 設定項目: サイドバーの設定項目
